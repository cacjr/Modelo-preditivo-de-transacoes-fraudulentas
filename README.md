Este projeto tem como objetivo desenvolver um modelo de previsão de fraudes em transações de crédito para uma instituição financeira. A proposta é identificar transações suspeitas a fim de reduzir prejuízos financeiros e aumentar a segurança e a confiança dos clientes. Para isso, utiliza-se uma base de dados de transações de crédito contendo diversas bibliotecas que auxiliam na análise e, machine learning e na construção do modelo preditivo.
